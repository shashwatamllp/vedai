## 🛠️ Quick Start

### 1. Prerequisites
- **Ollama Engine**: [Download & Install Ollama](https://ollama.com)
- **Python**: 3.8 or higher

### 2. Installation (Developer Mode)
Clone the repository and install dependencies in one go:
```bash
git clone https://github.com/shashwatamllp/vedai.git
cd vedai
pip install -e .
```

### 3. Usage
Run the intelligent chat session directly in your project folder:
```bash
# Start AI Chat with Project Context
vedai chat

# Run System Hardware Diagnostic
vedai doctor

# Force a specific model
vedai chat --model llama3:8b
```
