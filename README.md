**VedAI** (via Ved4u) is a next-generation, hardware-aware terminal assistant that brings the power of world-class LLMs (Qwen 2.5, Llama 3, Mistral) directly into your local development workflow. **Zero latency. Zero subscription. 100% Privacy.**

## 🚀 Why VedAI?
Most local AI tools are just chat-bots. **VedAI is an Intelligence Engine.** It scans your project, understands your architecture, and provides expert-level code assistance without ever sending your data to the cloud.

### ✨ Key Features
- 🧠 **Project Intelligence**: Recursive codebase scanning with `.gitignore` awareness.
- ⚡ **Hardware Optimization**: Real-time diagnostic of RAM/VRAM to select the optimal model.
- 🔒 **Absolute Privacy**: Everything runs locally on your machine. No API keys required.
- 🛠️ **Developer First**: Command-line native experience with premium Markdown rendering.

## 🛠️ One-Shot Installation

Copy and paste the command for your terminal to set up everything (App + Ollama + AI Models) in one go.

### 🪟 Windows (Recommended)
This will automatically detect your best drive, install Ollama, and set up the AI Agent.

**Option A: Command Prompt (CMD)**
```cmd
git clone https://github.com/shashwatamllp/vedai.git && cd vedai && python installer.py
```

**Option B: PowerShell**
```powershell
git clone https://github.com/shashwatamllp/vedai.git; cd vedai; python installer.py
```

### 🍎 macOS / 🐧 Linux
```bash
git clone https://github.com/shashwatamllp/vedai.git && cd vedai && pip3 install -e .
```
*Note: For Linux/Mac, please ensure [Ollama](https://ollama.com) is installed manually.*

---

## 🏗️ Usage

Once installed, use these commands to interact with your project:

```bash
# 1. Start AI Chat with Project Context
vedai chat

# 2. Run System Hardware Diagnostic
vedai doctor

# 3. Force a specific model
vedai chat --model qwen2.5-coder:7b
```
